Google Sheets Setup (one time, 5 min)
1. Open sheets.google.com → create a new sheet called "Polymath"
2. Go to Extensions → Apps Script
3. Delete the default code. Paste the script from the polymath_sheets_script.gs file (included in your download)
4. Click Deploy → New deployment → Web app
5. Set "Who has access" to Anyone → click Deploy
6. Copy the Web App URL → paste it above → Save
7. Done. Every task you tick now writes to your sheet instantly.
